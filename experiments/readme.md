# Experiments

The experiments are organized as follows:

- Type of experiment
    - description of the experiment (.md)
    - folder with experiment name
        - Input files (.xml)
        - Script to execute (.sh)
        - Output folder containing the results
        - Notes concerning the experiment (.md)
